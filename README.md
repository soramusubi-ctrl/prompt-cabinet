# Prompt Cabinet

使ったプロンプトを保存して、探して、もう一度使うための小さなWebアプリです。

## これは何？

AI画像生成や文章生成で使ったプロンプトを、端末内に保存して再利用するための道具棚です。

画像生成はしません。  
ログインもしません。  
APIキーも使いません。

## MVP機能

- プロンプト保存
- タイトル、用途、タグ、メモの登録
- 検索
- カテゴリ・タグ絞り込み
- ワンタップコピー
- お気に入り
- 削除
- PWA対応

## 保存について

データはブラウザの localStorage に保存されます。  
同じ端末・同じブラウザ内で使う前提です。

## 開発
