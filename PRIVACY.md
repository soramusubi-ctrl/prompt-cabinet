# Prompt Shelf プライバシーポリシー

最終更新日: 2026年6月3日

Prompt Shelf は、画像とプロンプトを管理するためのブラウザ拡張機能です。

## 1. 収集する情報

Prompt Shelf は、ユーザーの個人情報を収集しません。

氏名、メールアドレス、ログイン情報、閲覧履歴、Cookie、位置情報などは収集しません。

## 2. 保存されるデータ

ユーザーが入力または登録した以下のデータは、ユーザー自身のブラウザ内に保存されます。

- 画像
- プロンプト
- ネガティブプロンプト
- タグ
- 用途
- モデル名
- メモ
- お気に入り設定

これらのデータは、ブラウザの IndexedDB に保存されます。

## 3. 外部送信について

Prompt Shelf は、保存されたデータを外部サーバーへ送信しません。

外部 API、AI 生成 API、解析サービス、広告サービスには接続しません。

## 4. 第三者提供について

Prompt Shelf は、ユーザーのデータを第三者に販売、共有、提供しません。

## 5. データの削除

保存されたデータは、ユーザーが拡張機能内で削除できます。

また、ブラウザから拡張機能を削除した場合、保存データも削除されることがあります。

## 6. 本ポリシーの変更

本ポリシーは、必要に応じて更新されることがあります。

重要な変更がある場合は、配布ページまたはストア掲載情報でお知らせします。

## 7. お問い合わせ

お問い合わせは、Chrome Web Store の掲載ページに記載された連絡先までお願いします。
